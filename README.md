# 🏔️ Horizon Travel Concierge

**Horizon** is a premium travel landing page featuring an interactive AI concierge. Built with Vanilla JS, HTML, and CSS, it uses a custom state-machine to provide trip planning, airline recommendations, and travel FAQs. 

## 🌟 Features
* **Interactive Trip Planner:** Chat with Horizon to plan routes and get airline suggestions.
* **Smart Recommendations:** Curated highlights for major global destinations.
* **Modern UI:** Glassmorphism effects and soft animations.

## 🚀 Setup
1. Clone this repo.
2. Ensure `Pietro-de-grandi-T7K4aEPoGGk-unsplash.jpg` is in the same folder as `index.html`.
3. Open `index.html` in any browser.
